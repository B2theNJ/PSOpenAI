# Remove-OpenAIFile

## SYNOPSIS
Delete a file.

## SYNTAX

```
Remove-OpenAIFile
    [-FileId] <String>
    [-TimeoutSec <Int32>]
    [-MaxRetryCount <Int32>]
    [-ApiBase <Uri>]
    [-ApiKey <SecureString>]
    [-Organization <String>]
    [<CommonParameters>]
```

## DESCRIPTION
Delete a file.

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-OpenAIFile -FileId 'file-abc123'
```

Remove a file that has the ID `file-abc123`
