# Set-VectorStore

## SYNOPSIS
Modifies a vector store.

## SYNTAX

```
Set-VectorStore
    [-InputObject] <Object>
    [-Name <String>]
    [-ExpiresAfterDays <UInt16>]
    [-ExpiresAfterAnchor <String>]
    [-MetaData <IDictionary>]
    [-TimeoutSec <Int32>]
    [-MaxRetryCount <Int32>]
    [-ApiBase <Uri>]
    [-ApiKey <SecureString>]
    [-Organization <String>]
    [<CommonParameters>]
```

## DESCRIPTION
Modifies a vector store.

## EXAMPLES

### Example 1
```powershell
PS C:\> Set-VectorStore -InputObject 'vs_abc123' -Name 'NewName-123'
```

Modifies a name of vector store.
